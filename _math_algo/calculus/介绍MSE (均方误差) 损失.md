MSE（Mean Squared Error，均方误差）是机器学习中最经典的损失函数之一，特别常用于：

* 线性回归
* 神经网络回归任务
* LLM训练中的某些辅助任务（但语言模型主损失通常不是MSE）

---

# 1. MSE 解决什么问题

假设真实值：

```text
y = 10
```

模型预测：

```text
ŷ = 8
```

误差（Error）：

```text
error = ŷ - y
      = 8 - 10
      = -2
```

如果直接把误差相加：

```text
-2 + 2 = 0
```

正负误差会互相抵消。

所以需要：

```text
(error)^2
```

把负数变正数。

---

# 2. MSE公式

MSE=\frac{1}{N}\sum_{i=1}^{N}(\hat y_i-y_i)^2

其中：

* (N)：样本数
* (y_i)：真实值
* (\hat y_i)：预测值

步骤：

```text
预测值 - 真实值
↓
平方
↓
全部加起来
↓
取平均
```

---

# 3. 计算例子

真实值：

```text
[10, 20, 30]
```

预测值：

```text
[12, 18, 35]
```

### Step1：求误差

```text
[2, -2, 5]
```

### Step2：平方

```text
[4, 4, 25]
```

### Step3：求平均

```text
(4+4+25)/3

= 11
```

所以：

```text
MSE = 11
```

---

# 4. 为什么要平方？

平方有两个重要作用。

## 作用1：消除正负号

例如：

```text
error = -3
```

平方：

```text
9
```

不会被其它正误差抵消。

---

## 作用2：放大大错误

假设：

```text
error = 1
```

平方：

```text
1
```

而：

```text
error = 10
```

平方：

```text
100
```

大错误受到更严重惩罚。

| 误差 | 平方后 |
| -- | --- |
| 1  | 1   |
| 2  | 4   |
| 5  | 25  |
| 10 | 100 |

因此模型会优先修正特别离谱的预测。

---

# 5. MSE的几何意义

把每个预测看成一个点。

真实值：

```text
y
```

预测值：

```text
ŷ
```

两者距离：

```text
ŷ - y
```

MSE实际上是在计算：

> 预测点到真实点的平方距离平均值

在线性代数里可以写成：

```math
||ŷ - y||²
```

即：

```text
L2 Distance Squared
```

所以：

> MSE本质上是欧几里得距离（Euclidean Distance）的平方。

---

# 6. 为什么梯度下降喜欢MSE

因为MSE非常容易求导。

单个样本：

L=(\hat y-y)^2

对预测值求导：

\frac{\partial L}{\partial \hat y}=2(\hat y-y)

这个结果非常漂亮：

```text
梯度 = 2 × 误差
```

误差越大：

```text
梯度越大
```

模型更新越猛烈。

误差越小：

```text
梯度越小
```

模型更新越温和。

这正是梯度下降想要的性质。

---

# 7. 在神经网络中的样子

假设：

```text
输入 x
```

经过网络：

```text
ŷ = f(x)
```

然后：

```text
Loss = MSE(ŷ, y)
```

流程：

```text
Forward
    ↓
预测 ŷ
    ↓
计算 MSE
    ↓
反向传播
    ↓
更新参数
```

---

# 8. 与 MAE 对比

MAE（Mean Absolute Error）：

```math
MAE = (1/N) Σ |ŷ-y|
```

MSE：

```math
MSE = (1/N) Σ (ŷ-y)²
```

区别：

| 性质     | MSE | MAE    |
| ------ | --- | ------ |
| 大误差惩罚  | 强   | 弱      |
| 求导     | 平滑  | 在0点不光滑 |
| 对异常值敏感 | 高   | 低      |
| 训练稳定性  | 高   | 一般     |

因此神经网络中更常见：

```text
MSE
```

---

# 9. 在 LLM 中的关系

大语言模型训练主损失通常是：

**交叉熵损失（Cross Entropy Loss）**

而不是MSE。

原因是LLM做的是：

```text
预测下一个token
```

属于：

```text
分类问题
```

不是：

```text
回归问题
```

所以训练时一般：

```text
Logits
   ↓
Softmax
   ↓
Cross Entropy
```

而不是：

```text
Prediction
   ↓
MSE
```

---

# 从 LLM 学习角度如何理解 MSE

如果你已经学过：

* 向量
* 点积
* 梯度
* Attention

那么可以把 MSE 理解成：

> 一个把“预测结果与真实结果之间的距离”转换成单个数字的函数。

这个数字越大：

```text
预测越差
```

这个数字越小：

```text
预测越好
```

训练神经网络的本质就是不断调整参数，让这个数字尽可能接近 0。
